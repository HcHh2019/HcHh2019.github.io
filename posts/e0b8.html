Vue

Vue.js是一套构建用户界面的渐进式框架

Vue.js 只关注视图层， 设计时采用自底向上渐进式开发的构架，核心部件只关注用户界面的构建

Vue.js 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件

注意： Vue.js 不支持 IE8 及其以下 IE 版本

为什么要使用Vue

1.轻量级,体积小是一个重要指标,Vue.Js压缩后只有20多KB
2.移动优先,更适合移动端,比如移动端的Touch事件
3.易上手,学习曲线平稳,文档齐全
4.吸收了Angular(模块化)和React(虚拟DOM的长处),并拥有自己独特的功能
5.开源,社区活跃度高
…

当把鼠标放到HHHHH上时,鼠标下会有个标记

你看到的 v-bind 等被称为标记,指令带有前缀v-,以示他们是Vue提供的特殊特性

他们会在渲染的DOM上应用特殊的响应式行为。在这里,指令的意思为：将这个元素节点的title

特性和Vue实例的message属性保持一致。

如果打开浏览器的控制台，输入vue.message="哦哦哦哦哦",就会看到绑定了title特性的HTML

v-on

为了让用户和你的应用进行交互，我们可以用 v-on 指令添加一个事件监听器，通过它调用在 Vue 实例中定义的方法：

<div id="app_01">
    <p @click="func">{{message}}</p>
    <input type="button" value="事件绑定-点击" v-on:click="func">
    <input type="button" value="事件绑定-点击" @click="func">
    <input type="button" value="事件绑定-" v-on:monseenter="func">
    <input type="button" value="事件绑定-双击" v-on:dblclick="func">
</div>
<script>
    var vm= new Vue({
            el: "#app_01",
            data: {
                message: "ABC"
            },
            methods: {
                func() {
                    this.message = this.message.split('').reverse().join('');
                }
            }
    }); 
</script>

split() 方法用于把一个字符串分割成字符串数组
reverse() 方法用于颠倒数组中元素的顺序
join() 方法用于把数组中的所有元素放入一个字符串

注意在 reverseMessage 方法中，我们更新了应用的状态，但没有触碰 DOM——所有的 DOM 操作都由 Vue 来处理，你编写的代码只需要关注逻辑层面即可。

用Vue做简单计数器

    <div id="app">
    <button @click="add">+</button>
    <span>{{num}}</span>
    <button @click="jian">-</button>
</div>
<script>
    var vm= new Vue({
            el: "#app",
            data: {
                num: 0
            },
            methods: {
                add(){
                    if(this.num<10){
                        this.num+=1;
                    }else{
                        alert("最大为10!");
                    }
                },
                jian(){
                    if(this.num>0){
                        this.num-=1;
                    }else{
                        alert("最小为0!");
                    }
                }
            }
    });
</script>

v-show

根据表达式的真假切换元素的显示和隐藏

<div id="app">
    <button @click="q">点击隐藏图片</button>
    <img src="./1.jpg" v-show="isShow">
    <img src="./1.jpg" v-if="isShow">
    <img src="./1.jpg" v-show="age>20">
</div>

<script>
    var vm =new Vue({
        el: "#app",
    data: {
            isShow: true,
            age: 18
    },
    methods: {
            q(){
                this.isShow= !this.isShow;
            }
    }
    });
</script>

频繁切换的元素用 v-show,反之用 v-if

因为操作DOM数对性能的消耗比较大(v-if)

用Vue切换图片

<div id="app">
    <div>
            <img :src="img[index]" alt="">
            <a href="##" @click="sub" v-show="index!=0"><</a>
            <a href="##" @click="add" v-show="index<img.length-1">></a>
        </div>
</div>
<script>
    var vm = new Vue({
        el: '#app',
        data:{
                img: ['./1.jpg','./2.jpg','./3.png'],
                index: 0
        },
        methods:{
            add(){
                    this.index++;
            },
            sub(){
                    this.index--;
            }
        }
    });
</script>

用Vue做一个记录本功能

<div id="app">
    <header>
        <h1>记事本</h1>
        <input v-model="inputValue" @keyup.enter="add" autofocus="autofocus" autocomplete="off" placeholder="请输入">
    </header>

    <section>
        <ul>
            <li v-for="(item,index) in arr">
                <div>
                    <span>{{index+1}}.</span>
                    <label>{{item}}</label>
                    <button @click="sub(index)"></button>
                </div>
            </li>
        </ul>
    </section>

    <footer v-show="arr.length!=0" >
        <span>
            <strong  >{{arr.length}}</strong>
        </span>
        <button @click="clear">Clear</button>
    </footer>
</div>

<script>
    var vm = new Vue({
        el: "#app",
        data: {
            arr: [
                "ABC",
                "QWE",
                "ZXC"
            ],
            inputValue: ""
        },
        methods: {
            add(){
                this.arr.push(this.inputValue);
            },
            sub(index){
                this.arr.splice(index,1);
            },
            clear(){
                this.arr= []
            }
        }
    });
</script>
