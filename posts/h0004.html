scroll-out库

用ScrollOut库做的项目

Pages-A3
Pages-A4

ScrollOut可以检测滚动的变化，以显示，视差和CSS可变效果！

用npm i scroll-out -s或安装 下载.zip

ScrollOut是什么

ScrollOut是一个JavaScript微库，可检测浏览器中的滚动/调整大小变化，并为滚动元素和目标列表分配属性和实时CSS变量。

ScrollOut库不处理任何动画，但是为您提供了使用JavaScript动画库或仅CSS创建动画和过渡所需要的元素和工具！


一般流程为：

  1. 用data-scroll属性装饰目标
  2. 从JavaScript调用ScrollOut（）
  3. data-scroll当它们进入或离开视图时，每个都设置为进入或退出
  4. 随着元素滚动，它会翻转进出
  5. cssProps在配置中启用以添加实时CSS变量
  6. 用CSS或JavaScript对这些元素进行动画处理！

入门

从CDN安装

在文档的开头添加以下脚本

[html]
<script src="https://unpkg.com/scroll-out/dist/scroll-out.min.js"></script>

在加载/就绪或在文档底部的脚本中< body >，执行以下操作：

[js]
ScrollOut({
    /* options */
});

从NPM安装

从NPM安装卷展栏：

[sh]
npm i scroll-out -S

然后从包中导入ScrollOut并调用它

[js]
import ScrollOut from "scroll-out";

ScrollOut({
    /* options */
});

提示

使用CSS进行淡入[Fade In]

将此添加到您的CSS

[CSS]
[data-scroll] {
    transition: opacity 1s;
}
[data-scroll="in"] {
    opacity: 1;
}
[data-scroll="out"] {
    opacity: 0;
}

将此添加到您的页面以选择所有元素data-scroll

[html]
<script>
    ScrollOut();
</script>

使用JavaScript进行淡入淡出

[js]
ScrollOut({
    onShown: function(el) {
        // use the web animation API
        el.animate([{ opacity: 0 }, { opacity: 1 }], 1000);
    },
    onHidden: function(el) {
        // hide the element initially
        el.style.opacity = 0;
    }
});

与animate.css一起使用

使用Animate.css时，可以通过添加animated类来触发动画

[js]
ScrollOut({
    onShown(el) {
        el.classList.add("animated");
    }
});

强制animate.css重播

使用animate.css时，可能需要强制动画再次播放。幸运的是，有一种方便的方法可以强制浏览器重排文档并重播动画：

[js]
ScrollOut({
onShown: function(el) {
        // remove the class
        el.classList.remove("animated");

        // force reflow
        void el.offsetWidth;

        // re-add the animated cl
        el.classList.add("animated");
    }
});

定位滚动容器

要使用窗口以外的滚动窗格，请提供scrollingElement作为CSS选择器或元素

[js]
ScrollOut({
    scrollingElement: ".scrollable-pane"
});

创建一个粘性标题

下面的示例演示如何使用ScrollOut创建粘性标头

<html>
    <head>
        <style>
        .hero {
            height: 400px;
        }
        .header {
            position: relative;
            height: 100px;
        }
        .sticky-header {
            position: absolute;
            top: 0;
            left: 0;
            right: 0;
            height: 100px;
        }
        .sticky-header[data-scroll="in"] {
            position: fixed;
        }
        </style>
    </head>

    <body>
        <div class="hero"></div>
        <div class="header">
        <div class="sticky-header">HEADER</div>
        </div>
        <script>
        ScrollOut({
            targets: ".sticky-header",
            offset: 400
        });
        </script>
    </body>
</html>

与框架一起使用

大多数JS框架都有使用ScrollOut时应使用的安装/拆卸方法

Vue

export default {
    mounted() {
        this.so = ScrollOut({
        scope: this.$el
        });
    },
    destroyed() {
        this.so.teardown();
    }
};

API

Configuration

OptionsDescription
cssProps如果为true，则所有CSS变量都将添加到滚动元素和所有目标中。如果为false，则只会[ data-scroll ]在目标上修改该属性
offset当目标从屏幕顶部到达此距离（以像素为单位）时，目标变为可见。设置此选项将覆盖所有其他碰撞检测。这也可以是返回数字的函数
once元素只能从滚动到滚动一次更改。如果您只想一次转换所有元素，这将很有用。默认值为false。
scope使用此选项指定解析目标时要使用的根元素。如果未指定，则使用scrollingElemen
scrollingElement滚动容器。如果未指定范围，则假定所有目标都是该元素的子级。如果未指定，则使用documentElemen
targets元素的可选列表或CSS选择器。默认情况下，这是[ data-scroll ]
threshold在标记为可见之前必须可见的元素的比率。提供值0.2将要求元素的20％可见，然后再将其标记为可见。这也可以是返回数字的函数

Elements特殊选项

可以将以下属性添加到元素以启用选项：
scrollout-once:添加此选项可为单个元素启用一次选项


Events[事件]

可以将以下事件处理程序添加到ScrollOut构造函数中：

[ts]
ScrollOut({
    onShown: function(element, ctx, scrollingElement) {
        /* 显示元素时触发 */
    },
    onHidden: function(element, ctx, scrollingElement) {
        /* 隐藏元素时触发 */
    },
    onChange: function(element, ctx, scrollingElement) {
        /* 当元素更改可见性时触发 */
    }
});

ctx

PropertyDescription
offsetX距滚动元素左侧的距离
offsetY距滚动元素顶部的距离
elementWidth元素的宽度
elementHeight元素的宽度
indexScrollOut跟踪的元素与其他目标的索引
intersectX元素的位置与视口的位置。顶部= -1，视野= 0，1 =底部
intersectY元素的位置与视口的位置。左= -1，视野= 0，1 =右
viewportX元素相对于视口中心的水平位置。左= -1，居中= 0，且1 =右。这对于视差最有用
viewportY元素相对于视口中心的垂直位置。顶部= -1，居中= 0，并且1 =底部。这对于视差最有用
visible如果该元素在视口中可见，则等于1；否则，等于0。可以通过使用阈值选项进行调整
visibleX可见水平含量的比率。如果不可见则为0，如果100％可见则为1
visibleY可见垂直含量的比率。如果不可见，则为0；如果100％可见，则为1

Methods[方法]

ScrollOut有一些方法可以处理特殊情况。index()并且update()可以用于手动刷新DOM。teardown在构建单页应用程序时很有用

MethodDescription
.index()手动搜索元素。这适用于JS框架删除或插入DOM元素的情况
.update()手动检查元素是否已更新。这用于JS框架更改DOM的外观时
.teardown()如果您不再需要ScrollOut实例，请调用该teardown()函数

Scrolling Element[滚动元素]

滚动方向

ScrollOut在水平和垂直轴上检测方向。这对于创建上下滚动与向上滚动不同的CSS样式很有用

属性描述
data-scroll-dir-x当前滚动的水平方向-1、1或0。-1向上，而1向下。
data-scroll-dir-y当前滚动的垂直方向-1、1或0。-1左，1右。

变量描述
–scroll-dir-x当前滚动的水平方向-1、1或0。-1向上，而1向下。
–scroll-dir-y当前滚动的垂直方向-1、1或0。-1左，1右。

滚动百分比

ScrollOut计算滚动元素的总百分比，并将其作为CSS变量提供。这对于视差背景或其他效果很有用

变量描述
–scroll-percent-x水平滚动进度的比率。它从0开始到1结束
–scroll-percent-y垂直滚动进度的比率。它从0开始到1结束

滚动目标

目标尺寸

变量描述
–element-width元素的当前宽度
–element-height元素的当前高度

目标位置

变量描述
–intersect-x元素的位置与视口的位置。左= -1，视野= 0，1 =右
–intersect-y元素的位置与视口的位置。顶部= -1，视野= 0，1 =底部
–offset-x从滚动元素左侧开始的像素数
–offset-y从滚动元素顶部开始的像素数
–viewport-x元素相对于视口中心的水平位置。左= -1，居中= 0，且1 =右。这对于视差最有用
–viewport-y元素相对于视口中心的垂直位置。顶部= -1，居中= 0，并且1 =底部。这对于视差最有用

目标能见度

属性描述
data-scroll“进”或“出”。如果元素可见，则为“ in”，否则为“ out”。使用此属性装饰元素以自动定位这些元素

变量描述
–visible如果认为该元素可见，则值为1，否则为0。
–visible-x当前水平轴上可见内容的比率（0到1）
–visible-y垂直轴上当前可见内容的比率（0到1）
